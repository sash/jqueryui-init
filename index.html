jqueryui-init by sash

jQueryUI widget that lets you use data attributes to apply any of the jQueryUI widgets

Dependencies

jquery jquery-ui

Install

Include the jqeuryui-init.js script in your page after loading jquery and jquery-ui and you'r done :)
